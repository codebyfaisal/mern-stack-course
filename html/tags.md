### ✅ **HTML Tags**

#### **Document Metadata**

```html
<!DOCTYPE>  <!-- Document type declaration -->
<html>      <!-- Root of an HTML document -->
<head>      <!-- Container for metadata -->
<title>     <!-- Document title -->
<meta>      <!-- Metadata (charset, viewport, etc.) -->
<link>      <!-- Link to external resources -->
<style>     <!-- Internal CSS styles -->
<script>    <!-- JavaScript -->
<base>      <!-- Base URL for links -->
```

#### **Content Sectioning**

```html
<body>      <!-- Main document body -->
<header>    <!-- Page header -->
<nav>       <!-- Navigation links -->
<main>      <!-- Main content -->
<section>   <!-- Thematic grouping -->
<article>   <!-- Self-contained composition -->
<aside>     <!-- Sidebar or tangential content -->
<footer>    <!-- Page footer -->
<h1> to <h6>  <!-- Headings -->
```

#### **Text Content**

```html
<p>         <!-- Paragraph -->
<hr>        <!-- Horizontal rule -->
<pre>       <!-- Preformatted text -->
<blockquote><!-- Long quotation -->
<ol>        <!-- Ordered list -->
<ul>        <!-- Unordered list -->
<li>        <!-- List item -->
<dl>        <!-- Description list -->
<dt>        <!-- Description term -->
<dd>        <!-- Description definition -->
<figure>    <!-- Self-contained content -->
<figcaption><!-- Caption for <figure> -->
<main>      <!-- Main content -->
<mark>      <!-- Highlighted text -->
<time>      <!-- Date/time -->
<abbr>      <!-- Abbreviation -->
<address>   <!-- Contact info -->
<b>         <!-- Bold -->
<i>         <!-- Italic -->
<strong>    <!-- Strong importance -->
<em>        <!-- Emphasis -->
<small>     <!-- Smaller text -->
<s>         <!-- Strikethrough -->
<u>         <!-- Underline -->
<sub>       <!-- Subscript -->
<sup>       <!-- Superscript -->
<code>      <!-- Code snippet -->
<kbd>       <!-- Keyboard input -->
<samp>      <!-- Sample output -->
<var>       <!-- Variable -->
<q>         <!-- Inline quote -->
<ins>       <!-- Inserted text -->
<del>       <!-- Deleted text -->
<br>        <!-- Line break -->
<wbr>       <!-- Word break opportunity -->
```
